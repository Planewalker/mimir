I agree that gate isn't used as much as portal, but naming something "portal" or "the portal" doesn't have as much ring and, I feel, has more competition in the imagery department. 8 days later and we have a clear leader in the works : Planewalker's Codex
---

If we're good with that - we'll go forward with Planewalker's Codex as the official name. (Expect me to short hand it to Codex as I type )

Next task: Logo / masthead for the cover page. We're looking for something near professional grade if possible, and in color. (see the masthead at: http://www.firewall-darkcast.com/theeye ) To avoid copyright issues we need to avoid reusing the Lady's symbol if we can, and certainly avoid collages of professional work.

A good masthead would include the name, some sort of visual element to imply a logo concept, and be longer than it is tall (if it's on a cover it has to leave plenty of room below it for cover or content tables). This will also determine the overall visual tone of the 'zine - cooler colors / warmer colors, etc etc.

Detailed spec: color, 300dpi or greater, jpg, png, psd or bmp. Gifs don't handle the translation to PDF well. Please note any fonts used so they can be reused for other elements throughout the 'zine. Ditto color choices or texture elements used. (Basically, a psd file with all that information in your layers is going to be your best bet so we can develop further visual elements.)
