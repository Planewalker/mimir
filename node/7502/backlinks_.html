Ok. I see what's going on. It's *not* going to work that way. That link isn't a link to an image - it's a link to an actual *page*. Hang tight with me as I go into cyber-geek-dom

Note the end of this address: "detail?.dir=mail&.dnm=a974re2.jpg&.src=ph"
That's telling the yahoo system to generate an image and display it in the context of their page.

Don't use [img] to link to the image. Use [url].
